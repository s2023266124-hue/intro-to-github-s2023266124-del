# Intro to GitHub Assignment

## What I Learned
- Git is a version control system that helps track changes in projects.
- GitHub provides an online platform for collaboration and project management.
- The process of committing changes, pushing to a repository, and creating branches.
- The GitHub Flow: how to experiment safely without affecting the main project.

## Key Takeaways
- Understanding repositories as project folders containing files and version history.
- The difference between branches and forks, and when to use each.
- How to create pull requests and collaborate with others on GitHub.

## Challenges / Confusions
- Distinguishing between forking and branching in collaborative projects.
- Best practices for commit messages and pull request descriptions.
